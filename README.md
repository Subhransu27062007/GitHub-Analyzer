🔍 GitHub Analyzer

##📌 Overview

GitHub Analyzer is an AI-powered tool that evaluates a GitHub user's repositories and 
provides a summary of the repository such as What commits were made, What features were added, and so on. 
<br>
Along with the summary, the GitHub Analyzer also provides us the information about the people who contributed more to the repository in the form of bar graph.

##🎯 Purpose
To make things easier. 
<br>
When people are viewing a GitHub repository, it is sometimes difficult to track the commits made by the contributers. To address this issue, the GitHub Analyzer helps in summzrising the repositoy, saving users time in going thorugh whole things.
